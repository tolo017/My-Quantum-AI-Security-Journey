# The Hacker's Methodology

To think like a hacker, you must follow the process. Hacking isn't just one magic command; it's a series of logical steps.

## 1. Reconnaissance
Before touching the target, you gather intel.

## 2. Scanning
Now you look for doors and windows using tools like Nmap.

## 3. Exploitation
The moment of entry. Using an exploit to trigger a bug.

## 4. Post-Exploitation
Staying inside, escalating privileges, and cleaning up.
