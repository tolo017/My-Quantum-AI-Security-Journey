# Day 4: Networking Fundamentals II (Deep Dive)

Yesterday we saw how packets get to their destination. Today, we look at the content of those packets and how to secure (and break) the communication channels.

## Key Concepts
- **HTTP vs. HTTPS:** HTTP is plaintext; anyone on your local network can read it. HTTPS is encrypted using TLS.
- **SSH (Secure Shell):** The industry standard for remote administration, using public-key cryptography.
- **Wireshark:** A packet analyzer used to capture and inspect network traffic.
