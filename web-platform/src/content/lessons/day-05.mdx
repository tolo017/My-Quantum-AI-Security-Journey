# Day 5: Virtualization & Lab Isolation

Today, we master the art of "Lab Isolation." In cybersecurity, we often handle dangerous code or perform tests that could break a system. We use Virtualization and Containerization to create a "Sandbox"—a playground where we can break things without hurting our main computer.

## Key Concepts
- **Type-1 Hypervisor:** Runs directly on hardware (e.g., Proxmox, ESXi).
- **Type-2 Hypervisor:** Runs as an app on your host OS (e.g., VirtualBox, VMware).
- **Containers (Docker):** Shares the Host's Kernel. Extremely lightweight.
