# Day 9: Web Vulnerabilities I (OWASP Top 10)

Welcome to the world of Web Hacking! Almost everything is a web app today—from your bank to your thermostat. Today, we look at the most common ways they break.

## Key Concepts
- **SQL Injection (SQLi):** You trick a database into running a command it shouldn't by passing malicious inputs.
- **Cross-Site Scripting (XSS):** You inject a malicious client-side script (usually JavaScript) into a website, which runs inside another user's browser.
- **Command Injection:** You trick the web server's host system into running OS-level commands (like `ping` or listing directories).
